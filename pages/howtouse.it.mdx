# Onboarding

Guida per avviare chatgptitalia , tempo stimato =  5 minuti

### Step 1: Accedi

Accedi a questo sito utilizzando la tua email.

Devi prima creare la tua chiave API sul sito di OpenAI. 
Dopo esserti iscritto su OpenAI, visita il sito dello sviluppatore per creare una chiave personale. 
Questa chiave, una stringa alfanumerica, è necessaria per attivare questa app. 


Puoi ottenere la tua chiave API facendo login qui [qui](https://platform.openai.com/account/api-keys) poi andando alla pagina  [qui](https://platform.openai.com/login).

### Step 2: Memorizza la Chiave API

Incolla la tua chiave nell'apposito campo su ChatGPT Italia.

### Step 3: Inizia a Chattare

Ora sei pronto per iniziare a chattare. Clicca sul pulsante in alto a sinistra per creare una nuova chat.

### Step 4: Inserisci il Testo

Ci sono tre modi per inserire il testo: utilizzare l'input di testo normale, l'editor Markdown, o la digitazione vocale.

### Step 5: Utilizza i Prompt

Un prompt è un comando prestabilito che può specializzare la chat per uno specifico compito o carattere.

### Step 6: Riprendi la Chat da un Punto Specifico

Accanto a ciascun messaggio si trovano dei pulsanti. Se il messaggio è stato inviato da ChatGPT, puoi rigenerare quel messaggio. Se il messaggio è stato generato da te, puoi riprendere la chat da quel punto modificando il testo con l'editor.

### Step 7: Filtra le Chat

Se stai cercando una chat precedente, puoi filtrare la lista per titolo.

### Step 8: Utilizza i Prompt Preimpostati

Nella pagina principale, seleziona un prompt preimpostato per iniziare.
