# Come ottenere una chiave API OpenAI

Inizia visitando il [sito web ufficiale della piattaforma OpenAI](https://platform.openai.com/login). Segui i semplici passaggi per creare un account se non l'hai già fatto. In alternativa, utilizza le credenziali del tuo account Google o Microsoft esistente per accedere.

Una volta configurato il tuo account, vedrai l'icona e il nome del tuo profilo nell'angolo in alto a destra della home page della piattaforma OpenAI.

Per generare una chiave API:

1. Fai clic sul tuo nome nell'angolo in alto a destra per visualizzare un menu a discesa.
2. Scegli l'opzione "Visualizza chiavi API".

Verrai quindi indirizzato a una nuova pagina in cui troverai l'opzione "Crea nuova chiave segreta". Fai clic su questo pulsante se al momento non disponi di una chiave API. Non dimenticare di salvare immediatamente la chiave API poiché non sarà più accessibile dopo aver chiuso la finestra.

## Una chiave API OpenAI è gratuita?

Creare una chiave API OpenAI non ti costa nulla e ricevi un credito del valore di $ 5 (USD) come nuovo utente di prova gratuito (scade dopo tre mesi). Una volta scaduto o esaurito questo credito, dovrai inserire i dati di fatturazione se desideri continuare a utilizzare l'API. Anche senza inserire i dettagli di fatturazione, puoi comunque accedere al tuo account, anche se non potrai effettuare ulteriori richieste API.

Tieni presente che OpenAI applica limiti di velocità a livello di organizzazione. Questi limiti di velocità sono misurati in termini di RPM (richieste al minuto) e TPM (token al minuto).

Puoi trovare dettagli specifici sui prezzi in base al modello AI scelto (come GPT-4 o gpt-3.5-turbo) nella [pagina dei prezzi dei modelli AI di OpenAI](https://openai.com/pricing). Visita la [pagina Limiti di velocità di OpenAI](https://platform.openai.com/docs/guides/rate-limits) per comprendere i limiti di velocità esatti, esempi e altre informazioni utili.

## Come risolvere la "chiave API fornita non corretta" per OpenAI

Prima di presentare un reclamo a OpenAI, assicurati di aver effettuato i seguenti controlli:

* Scansiona la tua chiave API per individuare eventuali errori di battitura o spazio in eccesso
* Conferma se stai utilizzando una chiave API destinata a un'organizzazione o un progetto AI diverso
* Controlla se la tua chiave API è stata disattivata, eliminata o memorizzata nella cache

Seguendo queste istruzioni puoi potenzialmente risolvere rapidamente il tuo problema, risparmiando tempo e fatica.
